Asuran

A proposal of insurance on-demand, whereas it will check if the user is in high risk situation and it will notify and ask the user to enrol in a policy in insurance. It will check if the user is actually travelling a lot, in public and crowded area, or in area that are currently in war/riot or natural disaster. The data will be sent using MQTT platform to the mobile application.
